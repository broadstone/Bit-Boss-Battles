Bit Boss Battles (BBB) is an open source, viewer interactive bits widget for the Twitch platform. Streamers using BBB enable their viewers to fight each other using bits to become the next Bit Boss.

If you'd like to peek at the source, or report an issue, visit BBB's GitHub page.
